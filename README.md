# SME CyberGuard

A comprehensive cybersecurity dashboard for small and medium enterprises worldwide, providing real-time monitoring and threat protection across all devices.

## Overview

SME CyberGuard is a multi-tenant SaaS platform that enables business owners to monitor security threats, manage devices, and protect their networks in real-time. Built with modern web technologies, it offers a user-friendly interface for managing cybersecurity across distributed teams and locations.

## Features

- **Real-time Threat Monitoring**: Monitor security threats across all devices in real-time
- **Device Management**: Centralized dashboard for managing enterprise devices
- **Quarantine System**: Automatic isolation and management of malicious files
- **Web & USB Protection**: Block malicious websites and scan USB devices
- **Patch Management**: Track and manage software updates across devices
- **Multi-tenant Architecture**: Secure, isolated environments for multiple businesses
- **Global Accessibility**: Designed for businesses worldwide with scalable infrastructure

## Getting Started

To get started with SME CyberGuard development:

The only requirement is having Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

Follow these steps:

```sh
# Step 1: Clone the repository using the project's Git URL.
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory.
cd <YOUR_PROJECT_NAME>

# Step 3: Install the necessary dependencies.
npm i

# Step 4: Start the development server with auto-reloading and an instant preview.
npm run dev
```

## Tech Stack

This project is built with modern web technologies:

- **Frontend**: React + TypeScript + Vite
- **UI Framework**: shadcn-ui + Tailwind CSS
- **Database**: Turso (SQLite-compatible)
- **State Management**: TanStack Query
- **Charts**: Recharts
- **Icons**: Lucide React

## Deployment

SME CyberGuard can be deployed to various platforms:

- **Vercel/Netlify**: For static hosting with serverless functions
- **AWS/DigitalOcean**: For full infrastructure control
- **Docker**: Containerized deployment for scalability
